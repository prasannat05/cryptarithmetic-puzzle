# Cryptarithmetic Puzzle Game

- An interactive web-based puzzle game that enhances logical thinking through letter-to-digit mapping challenges.  
- Assign unique digits (0–9) to each letter so that the arithmetic equation is correct, ensuring no leading letter is zero.  
- Use keyboard or mouse to input values, and validate your solution in real time.  
- Includes hints and an AI solver to assist learning and exploration.  

Live Demo: https://prasannat05.github.io/cryptarithmetic-puzzle/
